# מייל — מדריך קצר

## פלטפורמה (אדמין BSD-YBM)

ניהול BSD-YBM → לשונית **מייל**.

1. **מתג גדול** — שליחת מיילים פועלת / כבויה (עוצר הכול בלחיצה).
2. **מה לשלוח** — סימונים פשוטים (מיילי מערכת, סיכום יומי, תזכורות מנוי וכו').
3. **שמור** + **שלח מייל בדיקה**.
4. **הגדרות נוספות** (אופציונלי) — כתובת שולח, הרצה ידנית.

מפתחות Resend/SMTP נשארים ב-Vercel בלבד.

## מנוי (מנהל ארגון)

הגדרות → **מיילים מהמערכת** — כל מנוי יכול לכבות/להפעיל לעצמו:

- סיכום יומי
- תזכורות מנוי ומעורבות
- מייל על התראות חשובות
- תזכורות גבייה ללקוחות

שערי הפלטפורמה חלים קודם; העדפות המנוי מצמצמות בלבד. מיילי אימות/הזמנה נשלחים לפי מדיניות הפלטפורמה.
