# Planner calendar

Shared month/day planner with meetings, tasks, reminders, and Jewish zmanim.

- Classic: used by `DashboardShell` calendar tab and `/m/dashboard/more/calendar`.
- OS later: import `PlannerCalendar` from `@/components/planner/PlannerCalendar` in `WidgetContent` next to (or instead of) `googleCalendar`. Prefer wrapping in `.dashboard-theme` for classic tokens, or leave default for dark OS chrome.
